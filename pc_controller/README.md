What it does:

Loads your CAN_schema.dbc
Connects to a CAN interface through python-can
Decodes frames using cantools
Shows a live Tkinter GUI with:
frame ID
message name
last-seen timestamp
raw bytes
decoded signal values
Logs unknown frames and decode errors

It will decode the messages in your DBC, including:

CONTROLLER_CMD
DEVICE_STATUS_NODE1 to DEVICE_STATUS_NODE4
DEVICE_CURRENT_NODE1 to DEVICE_CURRENT_NODE4

Install dependencies:

pip install cantools python-can

python can_dbc_gui_viewer.py --dbc CAN_schema.dbc --channel can0 --bustype socketcan
